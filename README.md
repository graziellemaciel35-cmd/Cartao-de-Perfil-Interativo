# Cartao-de-Perfil-Interativo
Cartão de apresentação visual desenvolvido com HTML e CSS puro. Possui faixa azul no topo, foto circular, dados pessoais, botões de redes sociais, ações e estatísticas. Totalmente responsivo e fácil de personalizar.

## Funcionalidades
- Faixa colorida superior
- Foto de perfil arredondada
- Ícones de redes sociais
- Efeito nos botões
- Layout adaptável
- Código simples e organizado

## Estrutura de Arquivos
 
 
pasta_do_projeto/
├── index.htm       # Estrutura principal
├── style.css       # Estilos visuais
├── profile.jpg     # Sua foto de perfil
└── README.md       # Este arquivo
 
plaintext
  

## Tecnologias
- HTML5
- CSS3
- Boxicons
- Fonte Poppins

## Como usar
1. Salve todos os arquivos na mesma pasta
2. Troque profile.jpg pela sua imagem
3. Abra index.htm no navegador

## Personalização
- Altere cores no style.css
- Edite textos e links no index.htm
- Ajuste tamanhos e espaçamentos
